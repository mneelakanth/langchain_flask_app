# langchain_flask_app

#### Clone this repository 
cd <folder_name>

#### Create a Virtual Environment
python -m venv <env_name>

#### Activate Virtual Environment
env\scripts\activate

#### Install Dependencies
pip install -r requirements.txt

#### Run Flask app
python main.py
